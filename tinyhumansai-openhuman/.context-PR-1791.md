## Impact Scan
### Changed function signatures / visibility

1. **`execute_tool_with_post_oauth_retry`** in `client.rs`: changed from `async fn` (private) to `pub(super) async fn`
   - Only new caller is `auth_retry.rs` (same module, `super` access) — correct visibility scope
   - No external callers affected

2. **`is_post_oauth_auth_readiness_error`** in `client.rs:477`: matching changed from **exact equality** (`normalized == *needle`) to **substring contains** (`normalized.contains(needle)`)
   - This is a **behavioral change** — now matches wrapped gateway messages like `"Action failed: Connection error, try to authenticate (gateway code 401)"`
   - This was previously only done by the now-deleted `match_retryable_auth_error` in `auth_retry.rs`
   - The constant `POST_OAUTH_AUTH_ERROR_STRINGS` values are **already lowercase** (`"connection error, try to authenticate"`), and the input is normalized to lowercase, so the contains check is correct
   - **Risk**: false positives if a different error happens to contain the substring — low risk given the specificity of `"connection error, try to authenticate"`

3. **`RETRYABLE_AUTH_ERRORS`** constant removed from `auth_retry.rs` — was `pub(crate)`
   - Grep needed: could any other file reference this?
   - The constant was only used within `auth_retry.rs` itself (by `match_retryable_auth_error`), so removal is safe

4. **`execute_with_auth_retry_inner`** in `auth_retry.rs`: signature unchanged but implementation completely rewritten
   - Now adds input validation (empty slug check) and constructs a JSON body before delegating
   - The `args: Option<serde_json::Value>` is now unwrapped to empty object if `None` — this matches the existing client pattern
   - Callers of `execute_with_auth_retry_inner` are unaffected (same signature)

### Removed exports
- `RETRYABLE_AUTH_ERRORS` — was `pub(crate)`, now removed. Need to verify no other files use it.
- `match_retryable_auth_error` — was private, safe to remove
- `is_retryable_auth_error` — was `#[cfg(test)]` only, safe to remove
