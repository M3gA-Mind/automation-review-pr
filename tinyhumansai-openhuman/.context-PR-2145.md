## PR Description
Adds the Polymarket integration (read + trading) as venue 1/3 for umbrella issue #1398 (Kalshi + Hyperliquid follow). Includes:
- **Read surface**: market/event/orderbook/price discovery via Gamma + CLOB public APIs, plus authenticated reads (positions, balance, open orders, USDC.e allowance) once an EOA is configured
- **Trading surface**: `place_order` + `cancel_order` against Polymarket CLOB with EIP-712 typed-data signing on Polygon (chain_id 137)
- **Approval gate**: every write action requires `arguments.approved == true` (stopgap until #1339)
- **Capability catalog split**: `skills.polymarket_readonly` (browse) + `skills.polymarket_trading` (Beta, Action)
- **Config**: additive fields on `PolymarketConfig` including `eoa_address`, `polygon_rpc_url`, `usdc_contract`, `clob_exchange_contract`, `derived_clob_credentials`
- **Tests**: 22 polymarket lib tests covering all 11 actions, approval gate, HMAC signing, EIP-712, retry/timeout, error handling
